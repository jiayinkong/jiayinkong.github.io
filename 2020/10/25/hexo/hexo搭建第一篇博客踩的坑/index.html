环境

首先检查电脑是否安装了nodejs和git，没安装需要先安装好。由于最近我升级了mac的系统，原本安装的git没有了。导致在使用hexo搭建博客的时候出现了问题。所以一定要注意安装了git没有。

我电脑上安装的版本：

mac安装git

方法一 官网下载安装包

去git的官网下载一个mac的git安装包，就是把那个dmg软件包下载下来，然后傻瓜式安装。

这个下载过程不是一般慢，耐心等待吧。

方法二 通过homebrew安装

先放上安装指令

brew install git

这个安装起来不太简单，之后有时间的话会写一篇关于怎么安装homebrew的文章。

安装完git后，需要在控制台输入如下指令配置下用户名、邮箱：

git config –global user.name “Your Name”

git config –global user.email “email@example.com“

用户名和邮箱我都是输入了github上的用户名和邮箱。

全局安装hexo-cli

sudo npm install -g hexo-cli

解决访问github慢的问题

如果访问github网站比较慢，可以尝试配置下本地的hosts。

访问站长工具

打开站长工具网页http://tool.chinaz.com/dns ，在那里输入github相关的地址，找到网络相对稳定的那个IP，配置到本地的hosts文件里。

我看很多文章也写了其他地址的配置，但我只配了这两个就可以了。

#安装成功后，nvm指令无法找到的时候，再在控制台输入这串：

export NVM_DIR=”$([ -z “${XDG_CONFIG_HOME-}” ] && printf %s “${HOME}/.nvm” || printf %s “${XDG_CONFIG_HOME}/nvm”)”[ -s “$NVM_DIR/nvm.sh” ] && . “$NVM_DIR/nvm.sh” # This loads nvm
